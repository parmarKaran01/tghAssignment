
# Assigment

A random quote generator app, Assigment for tgh tech



## Features

- Bookmark the quote
- Persist bookmarks with localstorage
- get random quote with selected tag
- mobile responsive


## Deployment
Deployed this app using Vercel

[Live app link](https://tgh-assignment-drab.vercel.app/)


## Tech Stack

**Client:** React, Redux, TailwindCSS
